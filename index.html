Introduction to C++

C++ is a middle-level programming language developed by Bjarne Stroustrup in 1979 at Bell Labs. It offers a blend of high-level and low-level features and is well-suited for developing hardware-operable software as well as large systems with sophisticated graphical user interfaces.

Example of C++ printing "Hello World!":

#include <iostream>
using namespace std;

int main() {
  cout << "Hello World!" << endl;
  return 0;
}

C++...

  • was developed by Bjarn Stroustrup
  • first appeared in 1985
  • was standardized by ISO and ANSI
  • supports procedural, object-oriented, and generic programming

Setting Up C++

Setting up C++ development environment depends on your operating system. Windows users typically use Visual Studio, while macOS users might use Xcode, and Linux users tend to prefer using GNU Compiler Collection (GCC).

Basic steps to setup C++ environment:

  • Install a text editor (e.g., VSCode, Atom)
  • Install C++ compiler (GCC for Linux, MSCV for Windows)
  • Configure your development environment to your liking

Basic Syntax

C++ syntax is robust and flexible, providing constructs that can map efficiently to typical machine instructions. It also supports data abstraction, object-oriented, and generic programming.

Basic syntax example (variables and output):

#include <iostream>

int main() {
  int num = 10;
  std::cout << "Number: " << num << std::endl;
  return 0;
}

Fundamental syntax elements in C++:

  • Statements and functions
  • Data types (int, float, double)
  • Conditionals (if, else)
  • Loops (for, while)

Control Structures

C++ provides all the typical control structures expected in a high-level language. These include conditionals, loops and case statements, which are used for creating versatile and dynamic algorithms.

Example of control structures(loops and conditionals):

for (int i = 0; i < 10; i++) {
  if (i % 2 == 0) {
    std::cout << i << " is even" << std::endl;
  }
}

Common control structures in C++:

  • if, else-if, else
  • while, do-while loops
  • for loops
  • switch-case statements

Functions

Functions in C++ help in modularizing the code. Each function can perform a specific task and can be called multiple times throughout the program which helps in reducing code redundancy and increases reusability.

Example function in C++ - Adding two numbers:

int add(int a, int b) {
  return a + b;
}

int main() {
  int sum = add(5, 3);
  std::cout << "Sum is: " << sum << std::endl;
  return 0;
}

Object-Oriented Programming Basics

Object-oriented programming (OOP) in C++ is a paradigm based on the concept of wrapping data and functions that work on the data within a single unit called an object.

Basic OOP example (classes) in C++:

class Car {
public:
  string color;
  string manufacturer;
  void drive() {
    std::cout << "The " << color << " car is driving." << std::endl;
  }
};

Core components of OOP in C++:

  • Class and Object fundamentals
  • Attributes and Methods
  • Constructors and Destructors
  • Access specifiers (private, protected, public)
